# Full-Stack Web App 

This project consists of a backend built with Node.js and Express.js and a frontend using Next.js. The backend handles API requests, while the frontend provides the user interface.

## Getting Started

### Enviroment variables

1. CLIENT_PORT: 3000
2. SERVER_PORT: 8080

### Prerequisites

## Clone Repository

```
git clone 'https://github.com/herveynavarro04/crispy-chainsaw.git'
git code crispy-chainsaw
```

## Install NodeJs

```
brew install node
```

## Install ExpressJs
```
cd server
npm install express
cd ..
```

## Install  NextJs
```
cd client
npm install next react react-dom
cd ..
```

### Running

## Frontend
```
cd client
npm install
npm run dev
```

## Backend
```
cd server
npm install
npm run start:dev
```
